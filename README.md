# 📉 Customer Churn Prediction

A Machine Learning project that predicts whether customers are likely to churn (leave a service) using Logistic Regression implemented from scratch.

This project includes:
- Data Analysis
- Feature Engineering
- Logistic Regression
- Gradient Descent
- Customer Churn Prediction
- New Customer Risk Prediction

---

# 📌 Problem Statement

The goal of this project is to predict whether a customer will:

- Stay with the company
- Churn (leave the service)

based on customer behavior and account-related features.

Target Variable:

```python
Churn
```

# 🧠 Features Used

| Feature | Description |

| Years | Years as customer |
| Num_Sites | Number of websites managed |
